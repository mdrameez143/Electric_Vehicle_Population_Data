# 🚗 Electric Vehicle Population Data Analysis

## 📌 Project Overview
This project analyzes Electric Vehicle (EV) population data to identify trends in EV adoption, vehicle types, manufacturers, model years, and geographical distribution. The analysis provides insights into the growth of electric mobility using data visualization and statistical techniques.

## 📂 Dataset
The dataset contains information about registered electric vehicles, including:

- VIN (Vehicle Identification Number)
- County
- City
- State
- Postal Code
- Model Year
- Make
- Model
- Electric Vehicle Type
- Electric Range
- Base MSRP
- Legislative District
- Electric Utility
- Census Tract

## 🎯 Objectives
- Analyze EV adoption trends over the years.
- Identify the most popular EV manufacturers and models.
- Compare Battery Electric Vehicles (BEVs) and Plug-in Hybrid Electric Vehicles (PHEVs).
- Study the geographical distribution of EVs.
- Create visualizations for better understanding of the dataset.

## 🚀 How to Run
1. Clone this repository.
2. Install the required libraries:
   ```bash
   pip install pandas numpy matplotlib seaborn
   ```
3. Open the Jupyter Notebook.
4. Run all cells to reproduce the analysis.

## 📌 Key Insights
- EV adoption has increased significantly in recent years.
- Battery Electric Vehicles (BEVs) dominate the dataset.
- Certain manufacturers account for the majority of EV registrations.
- Urban areas have higher EV adoption compared to rural regions.
